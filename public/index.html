Movies API - Documentación

🎬 Movies API

API RESTful para gestión de películas y géneros cinematográficos

📖 Descripción

Esta API permite gestionar una completa biblioteca de películas y sus géneros. Incluye funcionalidades avanzadas de autenticación con JWT, operaciones CRUD completas, filtrado múltiple y búsqueda inteligente por nombre, director y descripción.

🎬 Gestión de Películas

CRUD completo para películas con información detallada incluyendo director, año, géneros, duración, rating y más.

🎭 Gestión de Géneros

Sistema completo de géneros cinematográficos con descripción y popularidad para organizar las películas.

🔍 Búsqueda Avanzada

Filtrado por múltiples campos: género, año, rating, director, idioma y búsqueda de texto libre.

🔐 Autenticación JWT

Sistema seguro de autenticación con roles de usuario para proteger operaciones administrativas.

🔐 Autenticación

⚠️ Nota: Los endpoints de creación, actualización y eliminación requieren autenticación JWT.
Incluir el token en el header: Authorization: Bearer [tu_token]
POST /api/auth/register
Registrar nuevo usuario
POST /api/auth/login
Iniciar sesión y obtener token JWT

🎬 Endpoints de Películas

GET /api/movies
Obtener todas las películas con filtros opcionales
Filtros disponibles:
?genre=Action - Filtrar por género
?year=2023 - Filtrar por año
?rating=8 - Películas con rating mayor o igual
?director=nolan - Filtrar por director
?language=english - Filtrar por idioma
?search=batman - Buscar en título, director y descripción
?page=1&limit=10 - Paginación
?sort=-rating - Ordenar por campo (- para descendente)
GET /api/movies/stats
Obtener estadísticas generales de películas
GET /api/movies/:id
Obtener una película específica por ID
POST /api/movies
Crear nueva película (requiere autenticación de admin)
PUT /api/movies/:id
Actualizar película existente (requiere autenticación de admin)
DELETE /api/movies/:id
Eliminar película (requiere autenticación de admin)

🎭 Endpoints de Géneros

GET /api/genres
Obtener todos los géneros con filtros opcionales
Filtros disponibles:
?search=action - Buscar por nombre de género
?popularity=5 - Géneros con popularidad mayor o igual
?page=1&limit=20 - Paginación
GET /api/genres/:id
Obtener un género específico por ID (incluye sus películas)
POST /api/genres
Crear nuevo género (requiere autenticación de admin)
PUT /api/genres/:id
Actualizar género existente (requiere autenticación de admin)
DELETE /api/genres/:id
Eliminar género (requiere autenticación de admin)
